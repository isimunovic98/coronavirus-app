# CoronaVirusApp

## Description

This an application which will implement a simple Corona virus tracking app with Corona related news.


## Implementation

* UIKit
* TabBar
* MVVM-C architecture
* Combine framework
* user location
* Mapkit
* WebView, external browser
* pagination
* recoverable and unrecoverable errors
* loading animations
